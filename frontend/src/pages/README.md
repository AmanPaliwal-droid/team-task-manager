# Team Task Manager

A full-stack Team Task Management web application built using the MERN stack.

## Features

- User Authentication (Signup/Login)
- JWT-based Protected Routes
- Project Management
- Task Management
- Task Status Tracking
- Dashboard Overview
- MongoDB Database
- REST API Architecture
- Railway Deployment Ready

---

# Tech Stack

## Frontend
- React 18
- React Router DOM v6
- Axios

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
